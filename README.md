# scroller
Epitech, first year Rush. Graphic, make a scroller with different effect.

To compile : make To compile again and clean ".o" and the binary : make re. To clean all : make fclean.

To use it : 
              --> ./scroller text
              --> ./srcoller thick
              --> ./scroller mask
              --> ./scroller pokemon
              --> ./scroller slider
              --> ./scroller stars
