# Scroller

### Goal
###### The project : make a scroller with different effects !

### Program compilation and launch:
  ```sh
        make
```
  ```sh
        ./scroller text
        ./srcoller thick
        ./scroller mask
        ./scroller pokemon
        ./scroller slider
        ./scroller stars
```
